Note that you don't need a MySQL server for the RSS feed. Having said 
that, if you want to set up mysql, I'm guessing it will go something 
like this:
# yum install mysql-server (install mysql package. This will vary 
depending on your distro. For debian-based ones, including Ubuntu, use 
"apt-get install mysql-server")
# mysqladmin -u root -h localhost password "whatever_you_want" (set 
mysql root password)

If you don't mind having rdiffWeb use the mysql root password, then all 
you need to do at this point is re-run the rdiff-web-config script, and 
tell it to use "localhost" for the host, "root" for the username, and 
the password you gave it.

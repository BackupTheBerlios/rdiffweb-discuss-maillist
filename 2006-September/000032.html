I've been kicking around the idea of writing a setup script, that, given 
a freshly installed Ubuntu Server machine, would download and install 
the necessary software to run rdiff-backup and rdiff-web. The script 
would presumably prompt for a username, password, and root directory. It 
could optionally set up stunnel, so that all communication would be 
encrypted. Is this something that users would find helpful?

As a side-note, I've also been dreaming of a live-cd that could be run 
on a dedicated rdiff-backup box. It would reformat the entire harddrive 
to store backup repositories, and store enough settings so that the next 
time it's run it wouldn't need to prompt for settings. The idea is that 
the average user could pop the cd in a spare computer, and 10 minutes 
later have a machine that they could back up to.  It probably won't 
happen anytime soon, but it's a nice thought...

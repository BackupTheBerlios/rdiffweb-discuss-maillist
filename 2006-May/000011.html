The flyout menus for Previous Revisions are very handy, but they fail in 
two situations:

1) JavaScript is disabled or simply doesn't work.
2) The file has an extremely long list of revisions and you need an 
early revision. (It works, but it's not particularly convenient.)

Both of these problems could be solved if the Previous Revisions link 
would point to a meaningful page, perhaps much like the Restore Folder 
view, that would let you easily navigate the full list of revisions. 
While not a critical problem, I think it would improve the user experience.

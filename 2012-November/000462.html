Nice work Patrik. Please consider becoming the rdiffweb maintainer, I am 
sure Josh would be happy to give you the details if you were willing to

Did you add the capability to sort lists by column heading? If you have 
a big directory of files and you want to retrieve one of the most recent 
(which is often the case) you have to navigate down the screen to find 
it, if you could sort in descending last-revision order you could find 
it very quickly (also would be good if each user could set the default 
sort order i.e. name/size/last-revision ascending/descending).

Did you incorporate some fixes that are already well known (but not seen 
in 0.6.3)? See my rdiffweb-installer here:

Using relative paths will almost certainly not work. Try using absolute 
paths to the certificate:
SSLCertificate=/etc/rdiffweb/rdw.pem
SSLPrivateKey=/etc/rdiffweb/rdw.pem

If this doesn't work, try running the server interactively with 
"rdiff-web -d", and report any error messages.

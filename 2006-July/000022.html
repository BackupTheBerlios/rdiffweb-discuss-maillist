Yes. I was kinda hoping for a simple cron addition, that would run a
script managed by the frontend.
All the page would really need would be:

  * Remote Computer Username (maybe ssh key as well?)

part, and I would probably not bother with it for now, but it would be
very nice.

I'm more than happy to help with development of this, where ever

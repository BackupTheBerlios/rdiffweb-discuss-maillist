This is a python-specific problem.

Basically, python can't find the code for rdiff-web.

 

I've not seen this in any installation before. Can you confirm that you are 

a] running rdiff-web as root, 

b] the rdiff-web code is where you expect it (
/usr/lib/python2.6/site-packages/rdiffWeb )

c] root has full read/write permissions there

d] other programs (such as rdiff-backup) work

http://docs.python.org/tutorial/modules.html, that may also help.

 

When you fix this python problem, all versions of rdiff-web should work.

I am using rdiff-backup and rdiffWeb not only to back up my private 
files but also to securely share files over the Internet. That means I 
have two accounts. For one, I use the RSS feed to check whether my 
private files were backed up correctly. For the other, I use the RSS 
feed to tell me when new shared files might be available. (The 
repository listings are currently being maintained by the automatic 
spidering script, as described in 
http://www.rdiffweb.org/wiki/index.php?title=Automatic_Repository_Spidering.)

The problem, however, is that I am using Thunderbird as my aggregator, 
but it does not allow me to subscribe to multiple feeds at the same URL. 
I see several possibilities:

    * Use a different aggregator for the second feed (which is
      inconvenient, but would work)
    * Allow rdiffWeb users to have multiple "root" paths

Are there any other ways to solve this problem?

If this is an unusual setup, I'll live with the inconvenience of another 
aggregator. Otherwise, we should see if we can better favor these setups.

-Matthias Miller

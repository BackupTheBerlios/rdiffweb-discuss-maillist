given time. I could add this, though. As far as when it is done, it is 
done when the daemon is started, and then every interval after that.

One thing - the spider stops traversing a directory as soon as it sees 
that it is a repository, so the size of your repository shouldn't 
matter, only how many files/directories you have that are inside the 
root but not in a repository. Does this make sense?

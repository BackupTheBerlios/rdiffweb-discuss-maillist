Tonight I have been looking at rdiffweb and the first impression is that 
this is really something that I can use. But I found 2 little problems 
that might be related to versions of software of some sort, but that I 
want to discuss here anyway.

The first problem is the versioning of files. When there are more then 
one version of a file and I look at the previous versions list I see the 
current version and the previous versions but not the oldest one. I 
would expect to see the previous version until the first version in this 
list.

The second problem is when I have a folder that got deleted. When there 
has only been one version of the folder I don't get this version in the 
list if I want to restore the folder. If there are more versions, I get 
to see all the versions.

It looks as if something goes wrong in the code that calculates what 
versions to show in tables.

This are the relevant versions of software installed:

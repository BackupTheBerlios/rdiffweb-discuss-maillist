Apologies for the slow response.

The basics are as follows:

^variable$ : is replaced by the contents of variable. Variable must be a 
string.

<--StartDelete--> <--EndDelete--> : deletes everything in between, 
including markers

 <!--StartIncludeIf:expression-->  <!--EndIncludeIf:expression--> : 
deletes text in between markers if expression is false.  Always deletes 
markers

<!--StartRepeat:listName--><!--StartRepeat:listName--> : repeats text 
between markers for each item in listName. Items in listName should be a 
dictionary of names to values.


See rdiffWeb/templates/dir_listing.html and the corresponding python src 
file, rdiffWeb/page_browse.py for an example implementation of the above.
